## Dev tools
- VSCode
- nodejs-lts 20.10.0
- npm 10.2.3
- React.js
- Firebase

  - Hosting : 網站部屬
  - Extensions : 留言通知(Email)、文章搜尋(Algoliz)
- Semantic UI React
